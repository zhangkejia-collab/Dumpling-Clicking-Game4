# 🐍 學中文 · 任務闖關

一個專為手機設計的中文填空闖關遊戲，Google 貪食蛇風格介面，共兩個關卡，難度逐步提升。

## 🎮 遊戲關卡

| 關卡 | 名稱 | 說明 |
|------|------|------|
| Level 1 | ✍️ 生詞填空 | 句子中有一個空格，選出正確的生詞，共 6 題 |
| Level 2 | 📖 段落問答 | 閱讀三段短文，用中文簡答每個問題 |

- 完成 Level 1 才能解鎖 Level 2
- 每關完成後顯示通關畫面與星星評分

## 🚀 線上遊玩

👉 [點此開始遊戲](https://你的帳號.github.io/chinese-quiz-game/)

> 請將上方網址換成你自己的 GitHub Pages 網址

## 📁 檔案結構

```
chinese-quiz-game/
├── index.html   # 遊戲主體（單一檔案，無需額外依賴）
└── README.md    # 說明文件
```

## 🛠️ 部署到 GitHub Pages

1. 將此資料夾上傳到 GitHub repository
2. 進入 repository 的 **Settings → Pages**
3. Source 選擇 `main` branch，資料夾選 `/ (root)`
4. 點擊 **Save**，等待約 1 分鐘
5. 即可透過以下網址使用：

```
https://你的帳號.github.io/你的repo名稱/
```

## 🖥️ 本地測試

直接用瀏覽器開啟 `index.html` 即可，無需安裝任何套件或框架。
